# Digital Life Impact Dashboard

A modern, responsive analytics dashboard built with React, Tailwind CSS, and Recharts to explore the impact of digital life on health and productivity.

## Features

- Dark, professional dashboard UI
- KPI cards for core metrics
- Interactive filters by gender, age group, and profession
- Multiple chart types:
  - Line chart (screen time vs productivity trends)
  - Scatter plot (screen time vs sleep hours, stress as bubble size)
  - Bar chart (stress level by age group)
  - 100% stacked bar (eye strain vs sleep quality)
  - Heatmap matrix (screen time vs productivity distribution)
  - Profession comparison bar charts (screen time, social media, sleep, productivity)

## Tech Stack

- React
- Tailwind CSS
- Recharts
- Vite

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
